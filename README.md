# HandyParticles 

A fullscreen browser-based hand gesture experience using MediaPipe.
Single HTML file. No install. No build step.

## Live Demo
https://parshantb18.github.io/HandyParticles/HandyParticles.html

## How to Use
Open the link above in Chrome or Edge, allow camera access, and click Enter Experience.

## Modes
- ▶ Play — Hand skeleton, particles, pinch to zap, two-hand lightning & mandala
- ♥ Shapes — Heart, Peace sign, Star Burst using both hands
- ✏ Write — Air draw with index finger, pinch to lift pen, shake to clear

## Tech
Vanilla HTML/CSS/JS · MediaPipe Hands · Web Audio API · Canvas 2D
